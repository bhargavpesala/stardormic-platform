# Stardormic 🚀

Stardormic is an AI-powered startup network for students and builders in India.

## 🌐 Live Product
https://stardormic.com

## 💡 What it does
- Post startup ideas
- Find co-founders by skill and location
- Connect with builders and early-stage founders
- Build real projects from college
